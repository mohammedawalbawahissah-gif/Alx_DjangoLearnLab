## Blog Post Management Features

This blog supports full CRUD functionality for posts.

### Features
- View all blog posts
- View individual posts
- Authenticated users can create posts
- Only post authors can edit or delete their posts

### Permissions
- Create: Logged-in users only
- Update/Delete: Post author only
- Read: Public

### Technologies
- Django Class-Based Views
- Django ModelForms
- Authentication & Authorization Mixins
